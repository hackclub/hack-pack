# The *Hack Pack* auth server

We're using this to handle OAuth w/ GitHub as the first step in signing up for the Hack Pack.

## Running in development

```
$ git clone https://github.com/MaxWofford/hack-pack.git
$ cd hack-pack

$ npm i
$ npm run dev
```
